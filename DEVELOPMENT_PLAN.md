## 📝 提交规范

### Commit Message 格式

```
<type>(<scope>): <subject>

<body>

<footer>
```

**Type 类型:**
- `feat`: 新功能
- `fix`: Bug 修复
- `docs`: 文档更新
- `style`: 代码格式
- `refactor`: 重构
- `test`: 测试
- `chore`: 构建/工具

**示例:**
```
feat(system): 完成租户管理 CRUD

新增:
- Tenant Entity/Mapper/Service/Controller
- 租户注册接口
- 租户配额校验

测试:
- 租户创建测试
- 配额限制测试

Closes #1
```
